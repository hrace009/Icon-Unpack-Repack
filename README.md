# Icon Unpack Repack
 VB Script to Unpack & Repack DDS icon with a large of number files

# To Unpack
 Just drag & drop .txt file from dds icon to Extract.vbs . (eg. icon.dds & icon.txt)
 
# To Repack
 Just drag & drop folders Icon to REPACK2.vbs
